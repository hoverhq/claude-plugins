# Hover plugins for Claude Code

Official Claude Code plugins for [Hover](https://hoverfeed.com).

```
/plugin marketplace add hoverhq/claude-plugins
/plugin install hover@hover
```

## Plugins

- **hover** connects your Hover drone fleet to Claude Code (read-only): deployments, flights, recordings, detections, telemetry, and assets. See [plugins/hover](plugins/hover).
